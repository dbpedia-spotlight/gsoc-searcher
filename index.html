4. Wanna know how this has been done?





If you are curious about how we implemented this app, feel free to check our source code.  
    
    

  1. Type-ahead suggestion is done via DBpedia Lookup. This API takes in some phrase and searches the DBpedia knowledge base to find possible meanings for this phrase. The client side javascript uses AutoSuggest jQuery Plugin by Drew Wilson.

  2. Suggestion of related concepts is done via DBpedia's wikiPageLinks and using DBpedia Spotlight's notion of resource relatedness.

  3. Retrieval of projects is done via a SPARQL query over annotated projects. Projects were annotated with DBpedia Spotlight's Web Service. The resulting data was loaded to Virtuoso triple store, alongside wikiPageLinks dataset of DBpedia.

  4. Results are displayed by the DataTables jQuery plugin.
